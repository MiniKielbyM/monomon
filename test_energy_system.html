How to Use in Multiplayer

  1. Start the server: cd Server && npm start
  2. Open multiplayerTest.html in two browser windows
  3. Join a game with different usernames
  4. Drag energy cards from your hand onto your Pokémon
  5. Energy will appear as small icons on the Pokémon card
  6. Server enforces the "one energy per turn" rule
